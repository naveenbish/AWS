Creating a S3 Bucket in AWS



Steps to be Followed
  1. Search S3
  2. Click On S3
  3. Set one Unique name with lowercase Character and numbers.
  4. Object Ownership
    • Check ACLs enabled
  5. Block Public Access settings for this bucket
    • Uncheck Block all public access
    • check I acknowledge that the current settings might result in this bucket and the objects within becoming public.
  6. Create Bucket
  7. S3 Bucket Successfully Created
  8. Now Go to Properties and enable static web hosting. and then you will find link in Static web hosting Properties.
  9. Upload index.html file in Bucket.
  10. Reload your Hosted File link.
  11. Now You can see your static website.
    12. If you want to Access S3 using CLI.
