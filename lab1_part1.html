Agenda :

If we need to Store data of any instance or to restore data from S3 bucket then we used to create IAM role for S3 bucket and apply it to our instance.

Accessing S3 using CLI

    Step to be followed.
  1. Click on IAM
  2. Click on Roll
  3. Select EC2
  4. Select AmazonS3FullAccess
  5. Put Name of the Role and Create it.
  6. Select Instance.
  7. Click on Action
  8. Security
  9. Modify IAM role
  10. Select Your Created Role.
  11. Now go to your server "ssh -i pemkey ubuntu@'IP'.
  12. Install awscli in the server.
  13. Command used to install awscli on ubuntu.
    • wget https://awscli.amazonaws.com/awscli-exe-linux-x86_64.zip
    • apt install unzip
    • unzip awscli-exe-linux-x86_64.zip
    • sudo ./aws/install
    • aws --version
  14. aws s3 ls
  15. aws s3 cp filename s3://bucketname {Sending to Bucket}
  16. aws s3 cp s3://bucketname/filename . {Taking file to Server}
